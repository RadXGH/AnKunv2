# AnKunv2
AnKunv2 is an app a bit similar to YouTube but to stream anime.

Updated from [AnKun](https://github.com/RadXGH/AnKun) using Jetpack Compose.

# Tech Stack
- Kotlin
- AndroidX
- Jetpack Compose
- skrape{it}
- Coil
- Google Accompanist - HorizontalPager
